# 🧾 Invoice Management System

A full-stack invoice generator built with React, Node.js, Express, and MongoDB.

---

## 🚀 Features

- Create invoices
- Add multiple items
- Auto calculations (tax, discount, total)
- Save invoices to database
- Load & delete invoices
- Print / Export PDF

---

## 🛠 Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios
- react-to-print

Backend:
- Node.js
- Express
- MongoDB
- Mongoose

---
